# Panoramic Images Directory

Place your 360° equirectangular panoramic images in this directory.

## Required Images

The following images are referenced in `components/VirtualTourViewer.tsx`:

- `entrance.jpg` - Main entrance/lobby area
- `living-room.jpg` - Living room space
- `kitchen.jpg` - Kitchen/dining area
- `bedroom.jpg` - Master bedroom
- `balcony.jpg` - Private balcony/terrace

## Image Specifications

**Recommended:**
- Resolution: 4096 x 2048 pixels (2:1 aspect ratio)
- Format: JPEG
- Quality: 80-90%
- Projection: Equirectangular (spherical)

**Minimum acceptable:**
- Resolution: 2048 x 1024 pixels
- Format: JPEG

## Creating Panoramic Images

You can create 360° panoramic images using:

1. **Smartphone Apps:**
   - Google Street View (iOS/Android)
   - Panorama 360 (iOS)
   - Cardboard Camera (Android)

2. **Professional Cameras:**
   - 360° cameras like Insta360, RICOH THETA
   - DSLR with panoramic tripod head

3. **Online Tools:**
   - https://www.google.com/streetview/app/
   - Various 360° photo stitching software

## Tips for Best Results

- Use consistent lighting across all scenes
- Include distinctive features to help users orient themselves
- Avoid motion blur (use tripod when possible)
- Overlap scenes by 30-50% for smooth transitions
- Consider vertical FOV for ceilings and floors

## Placeholder Images

Until you add your own images, the app will show placeholder backgrounds.
Replace the placeholder filenames with your actual panoramic images.
