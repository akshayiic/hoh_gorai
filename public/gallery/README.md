# Gallery Images Directory

Place your standard photos for the image gallery in this directory.

## Required Images

The following images are referenced in `components/Gallery.tsx`:

**Interior:**
- `entrance.jpg` - Main entrance
- `living-room.jpg` - Living room
- `kitchen.jpg` - Kitchen area
- `bedroom.jpg` - Master bedroom

**Exterior:**
- `balcony.jpg` - Private balcony
- `building.jpg` - Building exterior

**Amenities:**
- `lobby.jpg` - Lobby area
- `pool.jpg` - Swimming pool

## Image Specifications

**Recommended:**
- Resolution: 1920 x 1080 pixels or higher
- Format: JPEG (quality 80-90%) or WebP
- Aspect ratio: 16:9 or 4:3

**Minimum acceptable:**
- Resolution: 1280 x 720 pixels
- Format: JPEG or PNG

## Image Organization

Images are categorized in the gallery by the `category` property in `components/Gallery.tsx`:
- `Interior` - Indoor spaces
- `Exterior` - Building exterior and outdoor areas
- `Amenities` - Shared facilities and features

## Tips for Best Results

- Use bright, clear images
- Maintain consistent aspect ratios within categories
- Optimize file sizes for web (under 500KB per image)
- Use proper lighting and composition
- Include both wide shots and detail shots

## Placeholder Images

Until you add your own images, the app will show placeholder backgrounds.
Replace the placeholder filenames with your actual photos.
